Step 1 — Create your Google Sheet
1
Go to sheets.google.com → Create a new sheet → Name it Gaavn Fresh Sales
2
In your sheet: click Extensions → Apps Script. Delete any default code, paste the script below, then click Save.
3
Click Deploy → New deployment → Web App. Set Execute as: Me, Who has access: Anyone. Click Deploy, copy the Web App URL.
